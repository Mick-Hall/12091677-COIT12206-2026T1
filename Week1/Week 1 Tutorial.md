# Week 1 Tutorial

Week one Tutorial was an introuduction to ***COIT12206 TCP/IP Principles and Protocols***, in which we met the unit coordinator, installed the supplied *GNS3* image and started our journey on *GNS3*.  

My tasks for the Tutorial were
- Create a new *GNS3* project 
- Add a single Linux host
- Add a box with some text to descibe the project
- Add some text to display the IP address
- Configure the node to use a static IP address from the 
- Start the node
- Open a web console and display the static IP address

I was a little behind due to the time taken to install the *GNS3* image, but once the tutor gave a demonstration and supplied the commands I found it relatively easy. 
I have minimal exposure to Linux and this week I learned the command to display the ip address - 'ip addr show'.
